9.1 束缚态微扰论

设体系的Hamilton量为 H^\hat{H}H^ （不显含 ttt ），能量本征方程为

H^ψ=Eψ \hat{H} \psi = E \psi H^ψ=Eψ

此方程求解一般比较困难，可以采用微扰论求解能量本征值与本征态的近似值。假设

H^=H^0+H^′ \hat{H} = \hat{H}_0 + \hat{H}' H^=H^0​+H^′

其中 H^0\hat{H}_0H^0​ 的本征值和本征函数比较容易解出，而 H^′\hat{H}'H^′ 是相对于 H^0\hat{H}_0H^0​ 的一个小量（ H^′≪H^0\hat{H}' \ll \hat{H}_0H^′≪H^0​ ），称为微扰，可以在 H^0\hat{H}_0H^0​ 的本征解的基础上，把 H^′\hat{H}'H^′ 的影响逐级考虑进去，以求出原方程尽可能精确的近似解。

将能量本征值与本征态逐级展开，即

ψn=∑s=0+∞ψn(s)=ψn(0)+ψn(1)+ψn(2)+⋯ En=∑s=0+∞En(s)=En(0)+En(1)+En(2)+⋯ \psi_n = \sum_{s=0}^{+\infty} \psi_n^{(s)} = \psi_n^{(0)} + \psi_n^{(1)} + \psi_n^{(2)} + \cdots \ \ \ E_n = \sum_{s=0}^{+\infty} E_n^{(s)} = E_n^{(0)} + E_n^{(1)} + E_n^{(2)} + \cdots ψn​=s=0∑+∞​ψn(s)​=ψn(0)​+ψn(1)​+ψn(2)​+⋯ En​=s=0∑+∞​En(s)​=En(0)​+En(1)​+En(2)​+⋯

⟨ψn(0)∣ψn(s)⟩=0(s=1,2,3,⋯ ) \langle \psi_n^{(0)} | \psi_n^{(s)} \rangle = 0 \kern 2em (s=1,2,3,\cdots) ⟨ψn(0)​∣ψn(s)​⟩=0(s=1,2,3,⋯)

依次称为零级方程、一级方程、二级方程……逐级求解，即可得到各级近似解。
